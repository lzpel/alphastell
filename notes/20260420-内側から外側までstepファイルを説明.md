# 内側から外側まで step ファイルを説明

`parastell/examples/` に生成される STEP ファイル群 (cad_to_dagmc example 実行時) の解説。ラジアルビルドは内側 (プラズマ側) から外側へ順に積み重なっている。

## 1. `chamber.step` — 真空チャンバー (プラズマ領域)

- **位置**: 一番内側、最終閉磁気面 (`wall_s=1.08`) の内側
- **中身**: 真空 / D-T プラズマ (mat_tag="Vacuum")
- **役割**: ここで核融合反応が起きる。中性子の発生源

## 2. `first_wall.step` — 第一壁 (FW), 厚さ 5 cm

- **位置**: プラズマに直接面する最初の固体壁
- **材料**: 通常タングステン (耐熱・耐照射)
- **役割**: プラズマからの熱流束と中性子フラックスを最初に受ける。**NWL (Neutron Wall Loading) はこの面で計算される重要境界**

## 3. `breeder.step` — トリチウム増殖ブランケット, 厚さ 25-75 cm (poloidal 変化)

- **位置**: FW の外側、**一番分厚い層**
- **材料**: 液体リチウム合金 (FLiBe, Li-Pb など)
- **役割**: 中性子を吸収して **トリチウムを増殖** (⁶Li + n → T + ⁴He)。同時に中性子エネルギーを熱に変換 → 発電。核融合炉の経済性を決める最重要構造物
- **なぜ厚みが変わるか**: プラズマの形が D 字型で場所により距離が違うため。例の matrix は 75 cm と 25 cm のパッチワーク → 「breeder を置ける場所は 75 cm、ダイバータ予定地など置けない場所は 25 cm」を模擬

## 4. `back_wall.step` — 背壁, 厚さ 5 cm

- **位置**: ブランケットの外側を閉じる構造壁
- **材料**: 構造鋼 (EUROFER, SS316 等)
- **役割**: ブランケット内の液体金属を閉じ込める容器壁。冷却配管が通る

## 5. `shield.step` — 中性子遮蔽体, 厚さ 50 cm

- **位置**: 背壁の外側、**2 番目に厚い層**
- **材料**: タングステンカーバイド、ホウ素入り水素化物、鉛など
- **役割**: ブランケットで吸収しきれなかった中性子を減速・吸収して **超伝導コイルを守る**。これがあって初めてコイルの寿命が成立する

## 6. `vacuum_vessel.step` — 真空容器 (VV), 厚さ 10 cm

- **位置**: 遮蔽体の外側、**トカマク全体の外壁**
- **材料**: SS316 等の構造鋼
- **役割**: 内部全体の超高真空を保持する圧力容器。**法的な「一次障壁」扱いで、事故時も構造を維持する最重要バリア**

## 7. `magnet_set.step` — 超伝導コイル群 (別管理)

- **位置**: 真空容器のさらに外側
- **材料**: 超伝導線材 (Nb₃Sn, HTS など) + 構造材ケース
- **役割**: ステラレータの磁場を作る。低温 (4 K) 動作なので **遮蔽体の性能がそのまま運転コスト**に直結
- **注意**: ラジアルビルドとは別の API (`construct_magnets_from_filaments`) で `coils.example` から生成される独立物体

## まとめ

中性子の旅路: **chamber で生まれる → first_wall で最初の衝突 (NWL) → breeder でトリチウムを作る (TBR) → back_wall は構造保持 → shield で減速・吸収 → vacuum_vessel は安全境界 → magnet_set への漏れ中性子が寿命を決める**

核融合炉設計の核心的トレードオフは「breeder (TBR 最大化) vs shield (コイル保護) の厚み配分」で、parastell の 2D thickness_matrix はまさにこれを場所ごとに最適化するための機能である。
